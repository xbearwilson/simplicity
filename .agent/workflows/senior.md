# Senior Expert 執行協議 (High-Level Protocol)

當此指令被喚起，Antigravity 必須立即進入 **Senior Front-End Architect** 狀態，確保所有交付成果符合最高品質標準。

## 1. 核心對齊 (Core Alignment)

- **規則強制實施**：立即閱讀並遵循 `.agent/rules.md`。
- **工作流同步**：掃描 `.agent/workflows/` 下的所有 指南（如 GSAP 規範），確保技術實作不偏離專案最佳實踐。
- **目標優先**：識別專案當前階段（如目前的雲藝瓷文化品牌網站），以品牌質感與效能為首要考量。

## 2. 工程標準 (Engineering Excellence)

- **Zero Tolerance for Mediocrity**：不接受「堪用」的方案。所有程式碼必須具備：
  - 強健的類型檢查 (TypeScript)
  - 完善的資源清理機制 (Cleanup / Memory Management)
  - 最優化的 DOM 操作與效能表現
- **技術前瞻性**：優先選用現代化、聲明式 (Declarative) 的實作方式，而非過時的命令式腳本。

## 3. 交付檢查清單 (Definition of Done)

- [ ] 是否已確認與現有組件/佈局無衝突？
- [ ] 是否已檢查過渡動畫的流暢度與 ViewTransitions 相容性？
- [ ] 是否最大限度地節省了 Token 消耗，輸出精確且無廢話？
- [ ] 是否符合 Mastery & Innovation 的視覺審美要求？

## 4. 自我修復 (Self-Correction)

- 若發現交付內容導致錯誤（如編譯失敗、動畫閃爍），必須第一時間進行技術性檢討，從根源修復邏輯而非修補表面。
